# Mini Quiz: Pipes and Pipelines

- [ ] user@ubuntu:~$ cat sample.txt || ./process.py
- [ ] user@ubuntu:~$ tr ' ' '\n' | sort | cat sample.txt
- [x] user@ubuntu:~$ cat sample.txt | tr ' ' '\n' | sort
